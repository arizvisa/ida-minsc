3. Modules¶

Each of the modules exposed by this project are suggested by the author to be thought of as a namespace that contains functions related to the module that they’re a part of. There are a number of modules which are exposed by default whereas others such as custom are available via the “custom” namespace.

3.1. Available modules¶

These modules compose the majority of this project and provide various tools that allow one to interact with the IDA database. These are each named according to the context they interact with.¶

  • 3.1.1. database – Database module
  • 3.1.2. enumeration – Enumeration module
  • 3.1.3. function – Function module
  • 3.1.4. instruction – Instruction module
  • 3.1.5. segment – Segment module
  • 3.1.6. structure – Structure module

3.2. Miscellaneous modules¶

These miscellaneous modules expose things that a user might care to interact with. Eventually some of these modules will be migrated into the base modules.¶

  • 3.2.1. ui – User Interface module
  • 3.2.2. tools – Tools module

3.3. Custom modules¶

These custom modules are accessed via the `custom` namespace and allow one to interact with some of the things that are provided by this project.¶

  • 3.3.1. tags – Tags module
  • 3.3.2. tagfix – Tagfix module
